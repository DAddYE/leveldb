# Leveldb

Experimental FFI bindings for leveldb

## Documentation

* Website [http://daddye.it/leveldb]
* Api Doc [http://daddye.it/leveldb/doc]

## Installation

    $ brew install snappy
    $ git clone git://github.com/DAddYE/leveldb.git
    $ cd leveldb
    $ rake compile
    $ rake console
